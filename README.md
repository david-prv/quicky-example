# quicky-example
A very, very small and simple demonstration of QuickyPHP.

## Installation

```bash
# get files
git clone https://github.com/david-prv/quicky-example.git
cd quicky-example

# install dependencies
composer install --no-dev

# run server
php quicky-cli start
```
